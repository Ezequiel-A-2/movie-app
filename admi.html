Agregar Peliculas

Nota: No tenemos una base de datos porque al momento de hacer esta pagina no obtuve el conocimiento para montar una y cargarle imagenes asique si bien esto funciona, hay que hacer una parte a mano (pero se le indicara cual), disculpe las molestias!

Explico cual es la parte manual: Como dije antes, no hay una base de datos asique para agregar una imagen se debe agregar la imagen a la carpeta local "resources" y luego en el campo "agregar portada" colocamos el nombre de la imagen que guardamos (ejemplo: si nuestra imagen se llama "space-jam.png" simplemente agregamos "space-jam.png" en el campo portada).
